# Product Docs

Здесь хранятся продуктовые спецификации, пользовательские проблемы, вехи и roadmap-материалы.

Текущие основные документы:
- `positioning.md` с позиционированием, Primary ICP и базовым framing продукта
- `landing-message-hierarchy.md` с иерархией сообщений для лендинга, hero, proof explanation и CTA-правилами
- `primary-personas.md` с primary persona, adjacent future personas и границами первого запуска

Опорные брендовые документы:
- `docs/brand/identity-package.md` с naming, voice и визуальным направлением
