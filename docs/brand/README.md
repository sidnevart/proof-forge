# Brand Docs

Здесь хранятся утверждённый identity package и рабочие бренд-операционные документы для ProofForge.

Основные документы:
- [identity-package.md](./identity-package.md) — утверждённый identity package и исходные решения по name / voice / visual direction
- [voice-system.md](./voice-system.md) — операционные правила бренд-голоса, паттерны фраз, анти-паттерны и UI copy examples
- [visual-system.md](./visual-system.md) — правила визуальной системы на уровне UI, signature components, state behavior, palette и typography constraints
- [naming-validation.md](./naming-validation.md) — checklist и scoring rubric для оценки будущих naming candidates

Использование:
- при написании копирайта сначала сверяться с `voice-system.md`
- при проектировании интерфейсов сверяться с `visual-system.md`
- при любых naming-sprint'ах использовать `naming-validation.md` как операционный фильтр
